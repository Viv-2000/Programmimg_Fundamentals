
# CineTicket System

## Overview
Welcome to the CineTicket System! This project is a simple, user-friendly Python application designed for cinemas. It acts as a virtual ticketing counter where box office managers or cashiers can manage ticket sales efficiently. This project is crafted to help beginners understand basic programming concepts through a real-world application.

## Features
- **Interactive Prompts**: The system prompts users to enter details like customer names, movie selections, and ticket types.
- **Dynamic Pricing**: Calculates total costs dynamically, applying discounts for reward program members and adding necessary fees.
- **Robust Data Management**: Utilizes lists and dictionaries to keep track of customers, movies, and transactions.
- **User-Friendly Menu**: Offers a straightforward menu system for navigating various functionalities, from buying tickets to adding new movie listings.
- **Advanced Capabilities**: In its final stages, the system supports batch ticket purchases and provides analytics on movie popularity and sales.

## Getting Started
To get started with the CineTicket System, make sure you have Python installed on your computer. You can download the project using the following command:
```bash
git clone <repository-url>
```

## How to Use
To run the system, move into the project directory and execute the script with:
```bash
python CineTicketSystem.py
```
Just follow the on-screen instructions to explore different features of the system through its menu.
